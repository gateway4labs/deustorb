# DeustoRB

DeustoRB is a simple client for the WebLab Deusto remote laboratory platform.

## Installation

Add this line to your application's Gemfile:

    gem 'deustorb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install deustorb
